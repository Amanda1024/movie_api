MyFlix API Documentation

Request URL HTTP Method Request Data Format Response Data Body Format
Get a list of movies /movies GET none A JSON object holding data about all the movies
Get data about a specific movie by title /movies/[title] GET none A JSON object holding data about a specific movie by title
Example:
{
title: 'Beauty and the Beast',
description: 'A prince cursed to spend his days as a hideous monster sets out to regain his humanity by earning love.',
rating: 'G',
released: '1991'
genre: 'Family',
genreDescription: 'A family film, is a film genre that contains children or relates to them in the context of home and family.',
director: 'Gary Trousdale',
directorBio: 'Gary A. Trousdale is an American film director, screenwriter, animator and storyboard artist, known for directing films such as Beauty and the Beast, The Hunchback of Notre Dame, and Atlantis: The Lost Empire. He frequently works with Kirk Wise and Don Hahn.',
birth: '1960',
imagePath: 'https://upload.wikimedia.org/wikipedia/en/7/7c/Beautybeastposter.jpg',
featured: 'false'
}
Get data about a movie genre /movies/genres/[name] GET none A JSON object holding data about which movies fall into a specified genre
Example:
{
genre: 'Drama',
movies: 'Dumbo'
}
Get data about the directors /movies/directors/[name] GET none A JSON object holding the data about what movies a specific director directed
Example:
{
director: 'Gary Trousdale,
movies: 'Beauty and the Beast'
}

Allow users to update their info /movies/users/[id] PUT none A message indicating that the username has been updated
Allow users to add a movie to their favorites /movies/users/[id]/favorites/[movieTitle] POST A JSON object holding data about the movie to add
Example:
{
title: 'Beauty and the Beast',
description: 'A prince cursed to spend his days as a hideous monster sets out to regain his humanity by earning love.',
rating: 'G',
released: '1991'
genre: 'Family',
genreDescription: 'A family film, is a film genre that contains children or relates to them in the context of home and family.',
director: 'Gary Trousdale',
directorBio: 'Gary A. Trousdale is an American film director, screenwriter, animator and storyboard artist, known for directing films such as Beauty and the Beast, The Hunchback of Notre Dame, and Atlantis: The Lost Empire. He frequently works with Kirk Wise and Don Hahn.',
birth: '1960',
imagePath: 'https://upload.wikimedia.org/wikipedia/en/7/7c/Beautybeastposter.jpg',

} 		A message indicating the movie was added to a user's list of favorites
Allow users to remove a movie from their list of favorites /movies/users/[id]/favorites/[movieTitle] DELETE none A message indicating that the movie was successfully removed from the user's list of favorites
Allow users to deregister /movies/users/[id] DELETE none A message indicating that the user account has successfully been deregistered
